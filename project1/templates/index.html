This the web application for CS50w Project 1.

You should be able to register for the website, log in, and log out. You should be able to search for books from a database of 5000 books by ISBN, title, or author. You should be able to leave a review for a book (score and text), and you should be able to see the average rating and number of ratings the book has from Goodreads.

Lastly, the website should be programmed to return a JSON response containing book info as part of an API.
